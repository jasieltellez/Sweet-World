Plugin para aceptar pagos de Webpay de Transbank en Chile usando WooCommerce mediante PayForm.cl

== Description ==
Plugin para agregar Transbank Webpay como medio de pago a WooCommerce mediante la plataforma de PayForm. El usuario será guiado paso a paso para configurar su plugin y estará aceptando Webpay Plus en Chile en solo minutos.

Este plugin además agrega características de recurrencia a los productos de WooCommerce de forma que sean compatibles con los pagos recurrentes de PayForm. En caso que el cliente agregue al carro de compras un producto recurrente, automáticamente se ajustará el carro para eliminar los productos no recurrentes. Los productos recurrentes son solo compatibles con tarjetas de crédito usando webpay Oneclick.

Se requiere de WooCommerce para que este plugin funcione.

== Installation ==
Una vez el plugin está instalado, será guiado paso a paso para configurar su tienda WooCommerce.
